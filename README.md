# Netflix Recommendation System 🎬

This project is a Netflix-style movie recommendation system built using Python and data analysis techniques.

## 📌 Project Overview
The system recommends movies/shows based on similarity between titles using machine learning techniques.

Recommendation systems like this are used by streaming platforms to suggest content to users based on patterns and similarities in data. 

## ⚙️ Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Google Colab

## 📊 Features
- Data preprocessing
- Movie similarity calculation
- Recommendation of similar movies
- Simple recommendation function

## 🚀 How to Run
1. Open the notebook file
2. Run all cells
3. Enter a movie name
4. Get recommended similar movies

## 📁 Files
- `netflix_recommendation_system.ipynb` – main notebook
- `dataset.csv` – dataset used
- `output.png` – sample result
